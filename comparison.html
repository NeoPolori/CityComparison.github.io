The Internet vs WWW

The Internet

The internet is a globally connected network system that allows for global communication and access to data resources via a vast network of personal, public, business, academic, and government networks. It is administered by organisations such as the Internet Assigned Numbers Authority (or IANA), which establishes global protocols.

World Wide World

The World Wide Web (WWW) is the dominant information retrieval service of the web (the global computer network). Online provides users with access to a vast array of documents that are linked to one another by hypertext or hypermedia linksâ€”that is, hyperlinks, electronic connections that link related pieces of data to allow a user easy access to them. The usage of hypertext allows the user to select a word or phrase from text and visit other documents that contain extra information on that word or phrase.
